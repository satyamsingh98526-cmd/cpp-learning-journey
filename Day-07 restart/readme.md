# Day-07 C++ Learning Journey 🚀

## 📚 Topics Covered

### 🔹 Decimal to Binary Conversion

Learned how to convert decimal numbers into binary using:

* division by 2
* remainder logic
* loops

#### Example:

13_{10} = 1101_2

### 🔹 Binary to Decimal Conversion

Learned how binary numbers are converted back into decimal using powers of 2.

#### Example:

1101_2 = 13_{10}

### 🔹 Two's Complement

Started understanding Two's Complement representation and how negative binary numbers work in computers.

Learned:

* bit inversion
* adding 1
* signed binary representation

## 💻 Practice Done
